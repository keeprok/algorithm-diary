# [level 2] 땅따먹기 - 12913

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12913)

### 구분

코딩테스트 연습 > 동적계획법(DP)

### 문제 설명

<p>땅따먹기 게임을 하려고 합니다. 먼저 땅은 총 N행 4열로 이루어져 있고, 모든 칸에는 점수가 쓰여 있습니다. 1행부터 땅을 밟으며 한 행씩 내려올 때, 각 행의 4칸 중 한 칸만 밟으면서 내려와야 합니다.</p>

<p>단, 같은 열을 연속해서 밟을 수 없습니다.</p>

<p>마지막 행까지 모두 내려왔을 때, 얻을 수 있는 점수의 최댓값을 return하는 solution 함수를 완성해 주세요.</p>

<h5>제한사항</h5>

<ul>
<li>행의 개수 N : 100,000 이하의 자연수</li>
<li>열의 개수는 4개이고, 땅(land)은 2차원 배열로 주어집니다.</li>
<li>점수 : 100 이하의 자연수</li>
</ul>

<h5>입출력 예</h5>

<table>
  <thead><tr>
    <th>land</th>
    <th>answer</th>
  </tr></thead>
  <tbody>
    <tr><td>[[1,2,3,5],[5,6,7,8],[4,3,2,1]]</td><td>16</td></tr>
  </tbody>
</table>

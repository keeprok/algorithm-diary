# [level 2] 2 x n 타일링 - 12900

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12900)

### 구분

코딩테스트 연습 > 동적계획법(DP)

### 문제 설명

<p>가로 길이가 2이고 세로의 길이가 1인 직사각형 모양의 타일이 있습니다. 이 타일을 이용하여 가로의 길이가 2이고 세로의 길이가 n인 직사각형을 만들려고 합니다.</p>

<p>n이 주어졌을 때, 이 직사각형을 채우는 방법의 수를 return 하는 solution 함수를 완성해주세요.</p>

<h5>제한사항</h5>

<ul>
<li>가로의 길이 n은 60,000이하의 자연수 입니다.</li>
<li>경우의 수는 항상 1,000,000,007 이하인 자연수입니다.</li>
</ul>

<h5>입출력 예</h5>

<table>
  <thead><tr>
    <th>n</th>
    <th>result</th>
  </tr></thead>
  <tbody><tr>
    <td>4</td>
    <td>5</td>
  </tr></tbody>
</table>

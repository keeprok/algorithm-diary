# algorithm-diary

프로그래머스 [코딩테스트 자바스크립트 편 100문제](https://school.programmers.co.kr/learn/challenges)를 기반으로, **하루 1문제**씩 풀며 기록하는 알고리즘 학습 저장소입니다.

## 학습 방법

면접에서 "어떻게 공부하나요?"에 답하기 위해, 아래 사이클로 반복합니다.

```
① AI(Cursor)에게 풀이 요청 → 전체 흐름·접근법 파악
        ↓
② 각 문제마다 주석으로 로직 작성 시도
        ↓
③-A. 주석 작성 성공 → 주석 기반으로 코드 직접 구현
③-B. 주석이 막히면 → Cursor에게 설명 요청 후 다시 익히기
        ↓
④ 코드 작성이 어려우면 → 함께 코드를 작성하며 이해
        ↓
⑤ Solo 재풀이 → 코드 삭제 → 주석만 남기고 처음부터 다시 풀기
```

**핵심 원칙**: 답을 외우지 않고, 주석으로 사고 과정을 설명할 수 있을 때까지 반복합니다.

Solo 제출 후 커밋 메시지에 `[solo]` 태그를 붙이면 README에 자동 반영됩니다. (`./scripts/install-hook.sh` 최초 1회 실행 시 커밋과 함께 갱신)
